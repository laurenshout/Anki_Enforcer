# Popup Assets

This folder contains images and other assets used for popup notifications in Anki Enforcer.

## Images

The `images/` folder holds the pictures displayed in motivational popups. The add-on randomly selects an image from this folder each time a popup appears.

### Adding Your Own Images

To personalize your popups:

1. Place image files (PNG, JPG, SVG) directly in the `images/` folder
2. Supported formats: PNG, JPG, JPEG, SVG, BMP
3. Images should be reasonably sized (under 1MB each recommended)
4. Restart Anki after adding images

If no images are present or they're unreadable, popups will display in text-only mode.

## Default Assets

The add-on ships with some default images to get you started. Feel free to replace or supplement them with your own.
